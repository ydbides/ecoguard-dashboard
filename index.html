About EcoGuard

EcoGuard is an autonomous aquatic monitoring and control system designed to detect instability in a water environment and respond automatically. This cloud dashboard reads live data from Firebase and makes the project accessible from anywhere in the world.

DS18B20 Temperature Sensor

Measures water temperature and supports automatic heater control and overheating detection.

Non-Contact Water Sensor

Detects whether water is present at the correct level without direct contact, helping prevent unsafe operation.

Turbidity Sensor

Estimates water cloudiness. Higher turbidity may suggest suspended waste, poor clarity, or declining water conditions.
Good: below 1800
Mild: 1800 to 2199
Bad: 2200 to 2999
Critical: 3000 and above

Fish Profile System

The dashboard can compare live conditions against popular tropical and monster fish temperature preferences. This makes the system adaptive to different species instead of using one fixed environmental rule for every fish.

Remote Monitoring

The ESP32 sends live data to Firebase Realtime Database. This GitHub Pages site reads that data and turns it into a proper dashboard instead of raw JSON.
