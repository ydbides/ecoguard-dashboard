About EcoGuard

EcoGuard is an autonomous aquatic monitoring and control system designed to detect instability in a water environment and respond automatically. This cloud dashboard reads live data from Firebase, supports user accounts, browser alerts, and smarter charts.

DS18B20 Temperature Sensor

Measures water temperature and supports automatic heater control and overheating detection.

Non-Contact Water Sensor

Detects whether water is present at the correct level without direct contact, helping prevent unsafe operation.

Turbidity Sensor

Estimates water cloudiness. Higher turbidity may suggest suspended waste, poor clarity, or declining water conditions.
Good: below 1800
Mild: 1800 to 2199
Bad: 2200 to 2999
Critical: 3000 and above

Fish Profile System

The dashboard compares live conditions against popular tropical and monster fish temperature preferences. Logged-in users can save their preferences in Firebase.

Live Browser Alerts

This version can show browser notifications when critical conditions appear while the dashboard is open. True background push notifications can be added later with Firebase Cloud Messaging.
